# Lab 11: Gesture Detection using APDS9960

## Objective

Detect left, right, up, and down hand gestures using the Nano 33 BLE Sense onboard APDS9960 sensor.

## Requirements

- Arduino Nano 33 BLE Sense and USB Micro-B cable
- Arduino IDE
- `Arduino_APDS9960` library, installed from Library Manager

## Files

- Sketch: [`code/gesture_detection_apds9960.ino`](code/gesture_detection_apds9960.ino)
- Evidence: [`images/gesture_detection_result.png`](images/gesture_detection_result.png)
- Demo: [`videos/gesture_detection_demo.mp4`](videos/gesture_detection_demo.mp4)

## Run

1. Install `Arduino_APDS9960` and upload the sketch.
2. Open Serial Monitor at 9600 baud.
3. Swipe above the onboard sensor and observe `LEFT`, `RIGHT`, `UP`, or `DOWN`.
